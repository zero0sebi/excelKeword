# 엑셀 키워드 카운터 (Excel Keyword Counter)

엑셀 파일 내의 특정 키워드 출현 빈도를 브라우저에서 즉시 집계하는 도구입니다. 모든 데이터 처리는 사용자의 PC(로컬)에서 이루어지며, 외부 서버로 데이터를 전송하지 않아 보안이 중요한 문서 처리에 적합합니다.

## 🚀 주요 기능

- **다중 파일 지원**: 여러 개의 엑셀 파일을 한 번에 업로드하여 통합 집계 가능
- **논리 조건 검색 (AND/OR)**: 
  - **AND**: 쉼표(`,`)로 구분하여 입력 시 모든 키워드가 포함된 셀 카운트
  - **OR**: 파이프(`|`)로 구분하여 복합 조건 검색 가능 (예: `A, B | C, D` -> (A AND B) OR (C AND D))
- **제외 조건 검색 (NOT)**: 제외 키워드 입력 시 해당 단어가 포함된 셀은 집계에서 제외
- **전체 분석 행 수 표시**: 분석된 모든 파일의 유효 데이터 행(Row) 총합을 표시
- **복수 조건 설정**: `+` 버튼을 통해 여러 종류의 검색 조건을 동시에 설정 가능
- **엑셀 결과 다운로드**: 집계된 결과를 다시 엑셀 파일(`.xlsx`)로 저장 가능
- **개인정보 보호**: 서버 통신 없이 브라우저 내에서 모든 분석 완료 (Serverless)

## 📁 파일 구성

1. **`keyword_search_local.html`**: 오프라인 전용 버전 (폐쇄망용)
   - 실행을 위해 동일 폴더에 `xlsx.full.min.js` 파일이 반드시 필요합니다.
2. **`keyword_search_cdn_v1.1.html`**: 온라인 전용 버전 (v1.1)
   - 인터넷이 연결된 환경에서 별도 파일 없이 이 파일 하나만으로 작동합니다.
3. **`xlsx.full.min.js`**: 엑셀 처리를 위한 필수 라이브러리 (SheetJS)
4. **`excelKeword_search_local.zip`**: 로컬 실행을 위한 전체 패키지 압축 파일

## 🛠 사용 방법

1. 사용 환경에 맞는 HTML 파일을 웹 브라우저(Chrome, Edge, Whale 등)로 엽니다.
2. **[엑셀 파일 업로드]** 버튼을 눌러 분석할 파일들을 선택합니다.
3. **[검색 키워드]** 입력란에 조건을 입력합니다.
   - **포함할 키워드**: `삼성, 반도체 | 현대, 자동차`와 같이 입력하여 복합 검색 가능
   - **제외할 키워드**: 결과에서 빼고 싶은 단어들을 입력합니다.
4. 더 많은 조건을 추가하려면 **`+`** 버튼을 활용합니다.
5. **[모든 파일 분석 시작]** 버튼을 클릭하면 분석 결과와 함께 **전체 분석 데이터 행 수**가 표시됩니다.
6. **[결과 엑셀 다운로드]** 버튼으로 결과를 저장할 수 있습니다.

## 🔒 보안 검토 보고서 (Security Report)

본 프로그램은 사용자의 민감한 데이터를 보호하기 위해 다음과 같은 보안 원칙을 준수합니다.

### 1. 데이터 처리 방식
- **로컬 처리**: 브라우저 표준 API인 `FileReader`를 사용하여 로컬 파일 시스템에서 브라우저 메모리로 데이터를 직접 읽어옵니다.
- **서버 통신 없음**: 사용자의 엑셀 내용이나 검색 키워드를 외부 서버로 전송하는 통신 코드가 존재하지 않습니다. (fetch, XMLHttpRequest, WebSocket 등 미사용)

### 2. 네트워크 보안성 확인
- **오프라인 작동**: `keyword_search_local.html` 버전은 인터넷 연결을 완전히 차단한 상태(폐쇄망)에서도 모든 기능이 정상 작동합니다.
- **검증 방법**: 브라우저 개발자 도구(F12) -> **Network(네트워크)** 탭에서 파일 분석 시 새로운 네트워크 요청이 발생하지 않음을 직접 확인할 수 있습니다.

### 3. 외부 라이브러리 안전성
- 본 도구에 사용된 `xlsx.full.min.js`는 세계적으로 검증된 오픈소스 라이브러리인 **SheetJS**의 배포 버전입니다.
- 라이브러리 내부의 URL 문자열은 엑셀 표준 규격(XML Namespace) 정의를 위한 용도이며, 실제 데이터 전송과는 무관합니다.
